# 第三阶段 Sprint 15：营销发布中心和一键转发包

## 1. 已实现范围

Sprint 15 将 Sprint 12–14 自动生成的营销素材交付给公司运营、销售、经销商和示范负责人使用，形成“审核素材 → 生成专属链接 → 复制或下载 → 人工发布”的工作流。

本阶段实现：

- 营销素材审核和发布状态。
- 手机端营销发布中心。
- 推广身份与 Django 登录账号绑定。
- 渠道专属追踪链接和二维码。
- 微信群、视频号、抖音文案及短视频脚本一键复制。
- 朋友圈海报和短视频封面单独保存。
- 完整 ZIP 一键转发包。
- 普通推广账号与管理员之间的权限隔离。

入口：`/marketing/`。

## 2. 素材状态流程

```text
公开阶段发布
      ↓
自动生成素材：待审核
      ↓ 管理员审核
    可发布
      ↓ 运营人员登记
  已登记发布
      ↓ 内容下线或停止使用
    已停用
```

状态规则：

- `待审核`：管理员可以预览和编辑，不能生成可用对外链接。
- `可发布`：销售和经销商可以查看、复制、生成链接和下载转发包。
- `已登记发布`：保持所有传播能力，用于表示内容已经在外部渠道发布。
- `已停用`：追踪链接不再进入阶段内容，普通推广账号不能查看。

管理员可以在“传播与二维码 → 营销素材包”列表批量执行：

- 审核通过：标记为可发布。
- 登记为已发布。
- 停用素材。
- 重新生成文案、海报和封面。

## 3. 推广身份和登录账号

`PromotionIdentity` 增加“登录账号”：

- 一个推广身份最多绑定一个 Django 用户。
- 一个用户最多绑定一个推广身份。
- 推广身份类型包括销售人员、经销商和示范点负责人。
- 取消“有效”后，该用户失去营销发布中心访问资格。
- 超级管理员和拥有 `campaigns.view_marketingpackage` 权限的后台账号也可以进入。

普通推广账号不需要设置为后台管理员，也不需要示范点采集权限。

配置步骤：

1. 在“认证和授权 → 用户”创建个人账号。
2. 在“传播与二维码 → 推广身份”创建推广人。
3. 选择推广类型、区域和登录账号。
4. 保持“有效”勾选并保存。
5. 将 `/marketing/` 和个人账号提供给推广人员。

## 4. 营销发布中心

### 4.1 素材列表

发布中心支持：

- 按品种或示范点名称搜索。
- 按生育阶段筛选。
- 按区域筛选。
- 管理员按素材状态筛选。
- 手机端紧凑卡片展示。

普通推广账号只显示“可发布”和“已登记发布”的素材。管理员可以查看待审核和停用素材，但不可为不可发布状态生成对外链接。

### 4.2 生成专属链接

进入素材详情后选择传播渠道：

| 渠道 | 来源代码 |
|---|---|
| 微信朋友圈 | `wechat_moments` |
| 微信群 | `wechat_group` |
| 微信视频号 | `wechat_channels` |
| 抖音短视频 | `douyin_video` |
| 抖音直播 | `douyin_live` |
| 田间二维码 | `field_qrcode` |

普通推广账号生成的链接强制绑定本人推广身份，不能通过修改表单参数替换成其他推广人。管理员可以选择公司公共链接或任一有效推广身份。传播渠道只表示“发到哪里”，销售、经销商等身份后续由后台用户组或绑定身份识别。

同一“素材 + 渠道 + 推广身份”重复生成时复用原追踪链接，避免产生大量重复链接。已经停用的同组合链接在重新生成时恢复有效。

### 4.3 一键复制

素材详情提供：

- 复制专属传播链接。
- 复制微信群文案。
- 复制视频号标题和文案。
- 复制抖音标题和话题。
- 复制短视频脚本。

优先使用浏览器剪贴板接口；不支持剪贴板接口时使用兼容方式复制。复制完成后按钮短暂显示“已复制”。

### 4.4 图片下载

- 朋友圈海报和短视频封面可以分别打开或保存。
- 图片继续存储在服务器 `MEDIA_ROOT\marketing\`。
- 素材缺少图片时页面显示提示，不返回损坏下载链接。

## 5. ZIP 一键转发包

准备专属链接后可以下载 ZIP，内容包括：

```text
朋友圈海报.png
短视频封面.jpg
微信群文案.txt
视频号标题和文案.txt
抖音标题和话题.txt
短视频脚本.txt
示范阶段链接.txt
专属二维码.png
```

文本使用带 BOM 的 UTF-8，方便 Windows 记事本和常见办公软件直接打开。

ZIP 在请求时动态生成，不在服务器长期保存，不需要增加定时清理任务。响应使用私有、禁止缓存设置。

## 6. 权限和安全

- 未登录访问 `/marketing/` 跳转负责人登录页。
- 没有营销权限且没有有效推广身份的账号会退出当前登录态，并返回营销发布中心登录页，方便切换账号。
- 普通推广账号只能查看可发布素材。
- 普通推广账号只能访问和下载绑定本人身份的链接与二维码。
- 修改 ZIP 或二维码 URL 中的链接 UUID，不能访问其他推广人的文件。
- 待审核和已停用素材不能生成对外追踪链接。
- 品种或示范点下线后，既有追踪链接停止跳转。
- 公开链接继续使用随机 UUID，不包含姓名、手机号和用户账号。

## 7. 管理员操作步骤

### 审核素材

1. 打开“传播与二维码 → 营销素材包”。
2. 按状态筛选“待审核”。
3. 检查文案、海报、视频封面和阶段页面。
4. 需要修改时进入详情编辑文案并保存。
5. 如果海报或封面显示“尚未生成”，点击页面右上角“立即生成/重新生成海报和封面”。
6. 勾选素材并执行“审核通过：标记为可发布”。

### 登记发布

1. 外部人工发布完成后勾选对应素材。
2. 执行“登记为已发布”。
3. 系统保存登记发布时间。

### 停用素材

1. 勾选不再传播的素材。
2. 执行“停用所选营销素材”。
3. 原追踪链接立即停止进入阶段页面。

## 8. 推广人员操作步骤

1. 手机打开 `/marketing/` 并登录个人账号。
2. 搜索品种或筛选阶段、区域。
3. 点击“打开转发包”。
4. 选择本次传播渠道。
5. 点击“生成专属链接和二维码”。
6. 选择复制文案、保存图片，或下载完整 ZIP。
7. 在微信、视频号或抖音中人工发布。
8. 客户通过链接或二维码进入后，访问和咨询自动归因到当前推广身份。
