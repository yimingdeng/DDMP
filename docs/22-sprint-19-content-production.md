# 第三阶段 Sprint 19：内容生产能力增强

## 1. 建设目标

本 Sprint 继续完善“示范表现自动生成营销素材”的内容生产能力，重点不是接入微信或抖音自动发布，而是让运营、销售和经销商拿到更适合直接传播的内容：

- 短视频脚本模板继续优化。
- 不同生育阶段生成不同话术。
- 朋友圈海报支持多版本。
- 生成经销商/推广人专属海报。
- 生成观摩会/看田邀请海报。
- 示范点阶段表现自动生成“本周重点推荐”海报。

对应需求：`MKT-CONTENT-PROD-001`、`MKT-POSTER-VARIANT-001`、`MKT-WEEKLY-RECOMMEND-001`。

## 2. 阶段化短视频脚本

系统按生育阶段使用不同内容重点：

| 阶段 | 话术重点 |
|---|---|
| 播种 | 播种基础、密度合理、规范管理 |
| 出苗 | 出苗整齐、苗势均衡、成苗基础 |
| 拔节 | 长势稳健、株型清秀、群体整齐 |
| 抽雄吐丝 | 花期协调、吐丝整齐、授粉基础 |
| 灌浆 | 灌浆充分、保绿性、抗病观察 |
| 成熟 | 站秆稳定、穗部整齐、脱水表现 |
| 收获 | 测产数据、商品性、收获表现 |

每条脚本包括：

- 开头：说明当前阶段最该看什么。
- 中间：提示拍摄镜头和表达重点。
- 重点：从阶段数据和核心标签中提取 3 个短标签。
- 结尾：引导查看完整示范、咨询、预约看田或申请试种。

## 3. 短视频选题拆分

营销素材包继续自动生成多条“单重点”短视频选题，每条包含：

- 选题标题。
- 单一重点。
- 口播脚本。
- 排序和启用状态。

营销发布中心可直接复制脚本，后台可编辑。

## 4. 多版本海报模板

新增“营销海报模板”记录，关联到营销素材包。

首版支持：

| 模板 | 用途 |
|---|---|
| 朋友圈图文海报 | 常规朋友圈和微信群传播 |
| 经销商专属海报 | 带经销商/推广人专属二维码 |
| 观摩会/看田邀请 | 成熟期、灌浆期和重点示范点组织看田 |
| 本周重点推荐 | 每周从阶段表现中挑重点内容传播 |

所有海报继续读取站点主题配置，紫色＋黄色主题可匹配棒中棒 889 线下物料风格。

## 5. 经销商/推广人专属海报

当营销人员在发布中心生成专属传播链接后，系统会生成一张专属海报：

- 标题包含推广人名称。
- 二维码指向该推广人的追踪链接。
- 客户扫码访问、咨询后继续回流到该推广身份。

该海报会出现在营销素材详情页，并进入 ZIP 转发包。

## 6. ZIP 转发包增强

下载完整 ZIP 时，除原有内容外新增：

- 朋友圈图文海报。
- 观摩会/看田邀请海报。
- 本周重点推荐海报。
- 经销商/推广人专属海报。

ZIP 内文件名会处理 `/` 等特殊字符，避免在 Windows 解压时生成异常目录。

## 7. 后台管理入口

后台新增或增强：

- “营销海报模板”可独立查看、搜索和编辑。
- 营销素材包编辑页可内联查看海报模板。
- 重新生成素材时会同步生成基础海报模板。
- 批量生成命令会补齐缺失的海报模板。

## 8. 数据库迁移与部署

新增迁移：

```text
campaigns.0005_marketingpostervariant
```

开发环境执行：

```powershell
cd D:\DDMP
.\.venv\Scripts\python.exe manage.py migrate
```

运行环境执行：

```powershell
cd D:\DDMP-RUNTIME\app\current
D:\DDMP-RUNTIME\venv\Scripts\python.exe manage.py migrate
D:\DDMP-RUNTIME\venv\Scripts\python.exe manage.py collectstatic --noinput
```

如果运行环境已有历史营销素材，可补生成：

```powershell
D:\DDMP-RUNTIME\venv\Scripts\python.exe manage.py generate_marketing_materials --missing-only
```

## 9. 验收清单

- [ ] 出苗、成熟等不同阶段生成的话术明显不同。
- [ ] 营销素材详情页显示短视频选题和可复制脚本。
- [ ] 重新生成营销素材后出现朋友圈图文、看田邀请和本周推荐海报。
- [ ] 生成专属链接后出现经销商/推广人专属海报。
- [ ] 专属海报二维码指向当前推广人的追踪链接。
- [ ] ZIP 转发包包含新增海报模板。
- [ ] 后台可查看和编辑营销海报模板。
- [ ] 批量补生成命令可为历史素材补齐缺失模板。

## 10. 当前边界

本 Sprint 不包含：

- 自动合成完整短视频文件。
- AI 自动配音和字幕。
- 微信或抖音自动发布。
- 自动抓取外部平台播放量。

这些能力需要单独评估平台权限、内容审核流程和视频转码成本。
