# 示范点列表与地图双模式

## 使用方式

- 示范点页面默认使用列表展示。
- 点击“地图展示”切换到地图，点击标记可进入示范点详情。
- 浏览器会记住上次选择的展示模式。
- 区域和省份筛选同时作用于列表与地图。
- 未配置坐标的示范点继续出现在列表中，地图下方会提示缺少坐标的数量。

## 后台配置

进入“示范点”编辑页面，在“位置”中填写：

- 地图纬度：`-90` 到 `90`，例如 `34.746600`。
- 地图经度：`-180` 到 `180`，例如 `113.625400`。

当前使用高德地图 GCJ-02 坐标。演示示范点使用城市中心附近的演示坐标，并不代表真实田块位置；正式发布前应替换为经过授权的田块坐标。详细地址是否公开仍由原有公开开关控制。

## 地图服务

当前地图使用高德地图 JavaScript API 2.0。需要在后台“站点配置 → 地图配置”填写高德 Web 端 JS API Key 和 `securityJsCode`，并在高德控制台配置开发地址或正式域名白名单。

开发环境采用前端安全密钥配置；正式公网环境建议按照高德官方安全方案改为服务端代理，避免安全密钥直接暴露。
