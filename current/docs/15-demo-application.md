# “我的示范”与示范申请

## 申请流程

1. 用户从平台首页进入“我的示范”，页面主体为负责人采集登录。
2. 没有账号的用户点击登录表单下方小字“申请成为示范户”。
3. 填写申请人、电话、品种、拟建示范点名称、所属区域、位置、计划面积、计划播期和示范经验。
4. 提交后页面提示等待联系，不提供线上进度查询。
5. 区域审核人只能查看本人负责区域的申请。
6. 审核通过时，区域审核人设置负责人登录用户名和至少 6 位的初始密码。
7. 系统创建负责人账号、草稿示范点，并把新账号绑定为该示范点负责人。
8. 区域后台人员通过电话通知审核结果和初始账号；申请人随后进入“负责人采集”。草稿示范点不会立即公开。

## 审核配置

- 使用现有“采集审核人”配置区域审核账号。
- 区域审核人的区域必须与申请选择的所属区域一致。
- 审核意见必填。
- 审核通过时用户名不能与已有账号重复，初始密码只保存安全哈希，不在申请页面明文展示。
- 退回不会创建示范点，用户可在“我的示范申请”中查看原因。
- 通过后生成的示范点仍需补充展示信息，并按原内容发布流程审核发布。
